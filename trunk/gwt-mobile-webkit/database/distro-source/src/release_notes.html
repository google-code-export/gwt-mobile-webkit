Release Notes for 1_5_0

DataService release

  • Added a DataService API to ease HTML5 Database applications
  • Improved Javadoc documentation, especially regarding the transaction steps
  • Added support for GWT.getUncaughtExceptionHandler()
  • Renamed jar file to gwt-html5-database.jar (was gwt-mobile-webkit-database.jar)
  • Fixed commandline scripts
  • Improved Ant scripts
  • Added Database.isSupported() method to determine whether the Database API is available
  • Fixed bug to gracefully handle SQL failures
  • Added comprehensive tests for Database and DataService API's

Known Issues

  • In some cases GWT2.0 (hosted and compiled mode) throws ClassCastException errors. This is especially the case if you use the ScalarCallback class. Add the -XdisableCastChecking option to the commandline (app gets faster and smaller all at the same time!)

Release Notes for 1.0.0

Initial release

  • Provides a GWT API to access the HTML5 Asynchronous Database features of Safari/WebKit and Opera.

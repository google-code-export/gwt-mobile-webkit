Release Notes for @API_VERSION@

Oversight fix: By error the property provider to determine 'gears' API was borked.

Release Notes for 0.9.2

Android fix release.

      Fixed the Geolocation API implementation so it works perfectly with all versions of Android. Due to the iframe design of GWT's default linker the gears_init.js code is applied to the iframe instead of the parent.
      'Hardened' the Geolocation methods - added try/catch blocks, to catch potential permission problems (Gears).

Known Issues

      API is (still) untested on Opera Mobile 9.7.
